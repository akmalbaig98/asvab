Class Job

source code

        Job

Single process job

Represents a single job, currently a script file to be run. Encapsulates this process and gets return value, stdout and sterr and writes these values to a file. Also keeps stats of the job.

Stats include:
  • script location
  • start time
  • end time
  • return value
  • whether the job is running
  • if the job succeeded
  • if the job is done
